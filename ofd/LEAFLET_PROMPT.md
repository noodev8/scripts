# Online Front Door — Leaflet AI Prompts

## Full Leaflet Prompt (for reference / quick drafts)

```
Design a print-ready A5 leaflet (front and back) for a service called "Online Front Door".

We build simple online presences for small local businesses — pubs, cafes, barbers, plumbers — that they can update themselves from their phone. No tech skills needed. We set it up, they keep it updated. £50 per month, cancel anytime.

The leaflet's purpose: we drop this off at local businesses to introduce the service. A separate handwritten note will include their personalised preview URL, so the leaflet itself should be generic — no individual business URLs on it.

Must include:
- The name "Online Front Door"
- Website: onlinefrontdoor.co.uk
- Price: £50 per month (cancel anytime) / £600 a year
- Placeholder for: name, phone, email
- The phrases: "No WordPress. No agency. No waiting."

The audience is non-technical local business owners, not tech people. The tone should feel local, trustworthy, and simple.
```

### Industry Variant Swaps

To create industry-specific versions, add one line to the prompt:

- Pub: `This version is specifically for pubs and bars.`
- Cafe: `This version is specifically for cafes and coffee shops.`
- Barber/Salon: `This version is specifically for barbers and salons.`
- Trades: `This version is specifically for tradespeople — plumbers, electricians, builders.`

## Canva Assembly Guide

1. Generate components 1, 2, and 7 above — that's enough for the generic version
2. In Canva, create an A5 portrait document
3. **Front panel:** drop in background (2), overlay phone mockup (1), add all text manually
4. **Back panel:** white/light background, add device mockup (7), type all copy manually
5. Use the AI-generated full leaflet (`Your-business-online2.png`) as your visual reference for layout, colours (navy/charcoal, amber/gold accents, white text), and typography choices
6. **Variants:** swap the background image and tweak headline — everything else stays the same

### Colour palette (from AI-generated version)
- Background: dark navy / charcoal
- Accents: amber / gold
- Text: white (on dark), dark charcoal (on light)
- Style: clean sans-serif throughout
