# Segment Meeting Rules

1. Every segment must have an active action — not "monitor", something testable
2. Owner reports: what did I try, what happened, what's next
3. "Numbers are the same" is fine. "I didn't do anything" is not.
4. GP trend + owner explanation + stock check. 5 mins per segment.
5. Compare like-for-like — seasonal drops are expected, surprises aren't.

## Action Ideas

- **Price:** Push a style up £3-5, or drop a slow mover to test volume
- **Stock:** Restock before it runs out, or send warehouse stock to FBA
- **Listing:** Better photos, titles, descriptions — especially Shopify
- **Range:** Add a missing colour, drop a dead one, test a new style with small stock
- **Promotion:** Feature in an email, adjust Google Shopping bids on a groupid

Each one is small, measurable, and done in a fortnight.
